About

Strawberry Creek Voices (SCV) collects and preserves community audio, interviews, and local stories. This GitHub Pages site serves as the project landing page, documentation hub, and quick reference for contributors.

Quickstart — publish to GitHub Pages

1) Create a repo named: strawberry-creek-voices (or your preferred name)
2) Add this file as index.html at the repository root
3) Commit and push to GitHub
4) In repository Settings → Pages, set source to:
   • Branch: main (or gh-pages)
   • Folder: / (root)
5) Wait a few minutes and visit: https://<your-username>.github.io/<repository-name>/

If you use a custom domain, create a CNAME file with the domain and add the DNS records as required by GitHub.

Contributing & Guidelines

Keep contributions focused and minimal: metadata (JSON/CSV), transcripts (plain text), and code for processing. Do not commit large binary audio files directly to the repo unless Git LFS or other storage is set up.

Maintenance notes

  • Use an audio normalization pipeline (sample rate, bitrate) before publishing.
  • Keep a clear metadata schema: id, title, date, contributors, location, license, transcript_link, audio_url.
  • Automate index generation with a script (scripts/build-index.py) that reads metadata and emits episodes.json.
